# LDesktop - 远程协助工具

基于 p2p（https://github.com/miroslavpejic85/p2p）的汉化优化版。

## 功能特点

- ✅ **P2P 直连** — A 电脑通过 IP:端口 直连 B 电脑
- ✅ **远程桌面** — 实时查看并操作对端桌面（鼠标键盘控制）
- ✅ **内嵌聊天** — 连接后可直接文字沟通
- ✅ **支持 NAT 穿透** — 内置 STUN 服务发现公网 IP
- ✅ **轻量便携** — 单 exe 运行，无需安装
- ✅ **安全加密** — AES-256 加密传输

## 使用说明

1. A 和 B 都打开 LDesktop
2. B 把自己的 **外网 IP**（显示在界面中）复制发送给 A
3. A 把 B 的 IP:端口填入 **远端地址**，点击 "连接远程"
4. 连接成功后，B 点击 "远程桌面" 即可让 A 看到并操作桌面

## 构建

```
dotnet build p2pconn.sln
```

或使用 GitHub Actions 自动构建（推送到 main 分支自动触发）。
